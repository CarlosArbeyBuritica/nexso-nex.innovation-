# 🚀 Nexso Next Innovation

**Nexso Next Innovation** es una plataforma moderna y tecnológica desarrollada con **Flask**, combinando diseño elegante y funcionalidad avanzada.  
Su objetivo es ofrecer innovación, estilo y experiencia premium a través de un catálogo inteligente y dinámico.

---

## 🖥️ Características principales

- 💡 **Diseño moderno:** colores negro y dorado con estilo tecnológico.
- 🔊 **Audio de bienvenida:** sonido personalizado al ingresar.
- 🛒 **Catálogo de productos:** organizado por categorías (Tecnología y Diseño).
- 🧑‍💻 **Panel de administración protegido:** gestión de productos, imágenes y pedidos.
- 📱 **Diseño responsivo:** optimizado para escritorio y dispositivos móviles.

---

## 🧩 Tecnologías utilizadas

- **Python 3** + **Flask**
- **HTML5**, **CSS3**, **JavaScript**
- **Bootstrap / Custom UI**
- **JSON / SQLite** para persistencia
